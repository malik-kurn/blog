Installasi CHR 7 (RouterOS)

Posted on Sep 5, 2024
tl;dr: Cara install CHR 7 (RouterOS)
Table of contents:
  • Langkah-langkah Installasi

Berikut adalah langkah-langkah untuk menginstall CHR (Cloud Hosted Router) versi 7.15.3 pada Mikrotik RouterOS menggunakan command line.

Langkah-langkah Installasi

  1. Unduh file image CHR menggunakan wget:

    wget https://download.mikrotik.com/routeros/7.15.3/chr-7.15.3.img.zip -O chr.img.zip

  2. Ekstrak file image CHR dan hapus file zip:

    gunzip -c chr.img.zip > chr.img && rm -f chr.img.zip

  3. Mount image ke dalam sistem:

    mount -o loop,offset=$((512*65570)) chr.img /mnt

  4. Dapatkan alamat IP dan gateway dari server:

    ADDRESS=`ip addr show eth0 | grep global | cut -d' ' -f 6 | head -n 1`
GATEWAY=`ip route list | grep default | cut -d' ' -f 3`

  5. Tambahkan konfigurasi IP dan route ke skrip autorun.scr di dalam image:

    echo "/ip address add address=$ADDRESS interface=[/interface ethernet find where name=ether1] /ip route add gateway=$GATEWAY " > /mnt/rw/autorun.scr

  6. Unmount image:

    umount /mnt

  7. Reboot sistem menggunakan perintah sysrq:

    echo u > /proc/sysrq-trigger

  8. Tulis image ke disk menggunakan dd:

    dd if=chr.img bs=1024 of=/dev/sda

Setelah langkah-langkah di atas selesai, CHR akan terinstall dan siap untuk digunakan. Pastikan untuk mengecek konektivitas jaringan setelah reboot.
