Bagi Anda yang mengalami kesulitan saat ingin mengubah versi PHP pada hosting namun hosting tersebut tidak menyediakan fitur Multi PHP Manager, Anda dapat mengatasinya dengan menambahkan kode pada file .htaccess.

Kode untuk Mengubah Versi PHP

Berikut adalah kode yang dapat Anda tambahkan ke dalam file .htaccess untuk mengatur versi PHP yang ingin digunakan.

Versi 5.1

<FilesMatch "\.(php4|php5|php3|php2|php|phtml)$">
SetHandler application/x-httpd-alt-php51___lsphp
</FilesMatch>

Cara Menggunakan

  1. Buka file .htaccess di direktori root situs Anda.
  2. Tambahkan kode yang sesuai dengan versi PHP yang ingin Anda gunakan.
  3. Simpan perubahan dan cek situs Anda untuk memastikan PHP berjalan pada versi yang diinginkan.

Catatan: Sebelum melakukan perubahan, pastikan untuk mencadangkan file .htaccess Anda.
